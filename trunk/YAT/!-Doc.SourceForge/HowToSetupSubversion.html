Install

1. Download "TortoiseSVN", the Subversion client for integration into Windows Explorer, from http://tortoisesvn.tigris.org/

2. Download "AnkhSVN", the Subversion client for integration into Visual Studio, from http://ankhsvn.tigris.org/

3. Install "TortoiseSVN" using the setup assistent

4. You're requested to reboot, do so

5. Install "AnkhSVN" using the setup assistent

 

Get repository working copy

1. In Windows Explorer, right-click on any folder

2. Select "SVN Checkout"
- URL of repository: https://svn.code.sf.net/p/y-a-terminal/code/trunk
- Checkout directory: Any directory on your local machine, avoid spaces in your directory path
