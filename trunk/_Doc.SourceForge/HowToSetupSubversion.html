Install

1. Download "Subversion" from http://subversion.tigris.org/

2. Download "TortoiseSVN", the Subversion integration into Windows, from http://tortoisesvn.tigris.org/

3. Download "AnkhSVN", the Subversion integration into Visual Studio, from http://ankhsvn.tigris.org/

4. Install "Subversion" using the setup assistent

5. Install "TortoiseSVN" using the setup assistent

6. You're requested to reboot, do so

7. Install "AnkhSVN" using the setup assistent

 

Get repository working copy

1. In Windows Explorer, right-click on any folder

2. Select "SVN Checkout"
- URL of repository: https://y-a-terminal.svn.sourceforge.net/svnroot/y-a-terminal/trunk/
- Checkout directory: Any directory on your local machine, avoid spaces in your directory path
